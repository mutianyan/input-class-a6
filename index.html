  • Instructions for using the app

    Use the keyboard keys to navigate the calculator keypads and the four models of switch controls. The default model is the five-key model.

    Models Available keys
    One-key [with interval 1.5s] e
    Two-key e, d
    Three-key e, d, a
    Five-key e, d, a, w, s
    • key w - up
    • key s - down
    • key a - left
    • key d - right
    • key e - select


  • Test results

  • Anything else you have tried out

    I compared the input time of using the one-key model with the three-key model, separately on a long sequence formula ”12+345-67*8/9=“ and a short one “987-65-42=”. The three-key model takes a significantly shorter time. Because it is extremely hard for the input sequences that do not follow the default order of navigation (from left to right, top to bottom, and wrap around). Users for that case have to wait for a clock circle just wanting the cursor to turn back around such as the case that the user presses 8 then wants to press 7.
